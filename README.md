# Money--Manager--Tutorial--cpp
## Description

Money Manager Tutorial is a beginner-friendly C++ console program designed as an interactive walkthrough for setting up a basic user profile. This program guides the user through choosing a name, selecting a city, making a breakfast decision, and selecting grocery items while enforcing rules such as item limits and duplicate prevention.

This project focuses on practicing core C++ concepts such as input handling, vectors, maps, helper functions, validation logic, and safe data access.


How to run

1. Clone or download the repository
2. Open the '.sln' file in Visual Studio
3. Make sure the project is set as the startup project
4. Build the solution (ctrl + f5)
5. Run the program (ctrl + f5)



Technologies used: 
C++
Visual Studio
Standard Template Library(STL)
'vector'
'unordered map'
